Project Overview

Each month, I download my credit card statements as .csv files and run them through a locally hosted, open source large language model to categorize transactions into a small set of buckets (Rent, Utilities, Household/Groceries, Dining Out, Travel, Subscriptions, and Miscellaneous). I then review the results to correct edge cases, such as ambiguous merchants or one-off purchases.

This workflow keeps my financial data private because nothing is uploaded to the cloud or sent to closed-source LLMs. The dashboard summarizes the current month at a glance and highlights category trends over the last six months, making patterns easier to spot early.
